<h1>TF_Forex</h1>

Forex Program to buy internation currencies (including USD, GBP, KES and EUR).

<h2>Getting Started</h2>

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

Prerequisites - localhost/virtual machine to be setup

Download project source code to your localhost and open your localhost in you browser.

URL should be: <localhost_name>/tf_forex/index.php

Example: localhost.tf_forex/index.php
